# Mini SaaS Template Store

A full-stack web application built as part of a Full Stack Web Developer Intern technical task.

## Features
- User Registration & Login (JWT Authentication)
- View available templates
- Mark templates as favorites
- View favorited templates
- Protected routes and logout

## Tech Stack

### Frontend
- React (Vite)
- React Router DOM
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs
